# Rime 鼠鬚管配色主題產生器

## 使用
☞ [Rime 西米 for Squirrel](https://gjrobert.github.io/Rime-See-Me-squirrel/)，單頁式應用程式

都設定好後，將最右欄「產生設定碼」下方全部複製起來，再放到 squirrel.custom.yaml 中。
